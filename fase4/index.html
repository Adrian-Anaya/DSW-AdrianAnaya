Introducción

Interfaz de Usuario

La interfaz de usuario se define como un medio por el cual un usuario interactua
con algun dispositivo tecnologico. Abarcando todos los puntos de contacto entre
la persona y el equipo.


Interaccion Humano-Ordenador

La interaccion humano-ordenador se puede identificar como la disciplina que
estudia el intercambio entre personas y humanos. Este tiene como objetivo el
intercambio de datos eficientes, minimizando los errores e incrementando
la satisfaccion del usuario.
